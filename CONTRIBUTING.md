# Contributing

This repository intentionally supports one hardware target only:

- Seeed Studio XIAO ESP32S3
- Ebyte E22P-868M30S
- MeshCore repeater role

Bug reports and focused pull requests for this target are welcome. Include the
firmware version, wiring, power supply, radio settings, serial log and exact
steps to reproduce the problem. Do not post passwords, private keys, node
identities, Wi-Fi credentials or packet logs containing private data.

Large protocol changes should be proposed to the upstream
[MeshCore project](https://github.com/meshcore-dev/MeshCore).
